2.2 - Embedding Javascript Expressions And Statements Within JSX

1 - Expressions In JSX

JSX will only accept expressions between curly braces {}. The curly braces tell our transpiler to process what is between the curly braces as normal JavaScript. Only JSX accept a return value we can not write a full statement Here are a few common types of expressions you may use:

# A few common types of expressions :-

  • Variable and object values
  • Function calls and references
  • Ternary Operators and ShortCircuit Operator Expressions
  • Loops and iteration

We have only used HTML tags as a part of JSX. But JSX becomes more useful when we actually add JavaScript code inside it. We will learn how to embed variables, refer to object property values, functions like normal and arrow functions or call functions and also we can place expressions like add mathematic operation, and use ternary operators or if-else and Iteration loops like map and foreach methods. So let start learning these things.

2 - JavaScript Expressions - Variables, Object Values and Reference

We can define and initialize our variables then use or call them inside a JSX {}. Also we can define an object that has properties that we can access or refer thru object variable name. Here's a valid things you can have in a JSX Expression.

# The valid expressions that we can use within JSX :-

  • A string like "hello"
  • A number like 10
  • An array like [1, 2, 4, 5]
  • An object property that will evaluate to some value

- Variable String and Arrays with Mathematic Expressions

# Example 1 - Defining a variable called UserName and insert it inside the { } :-

➤ An Example


  import React from "react"

3 - Function Calls, Conditional Statements, Ternary operator and ShortCircuit Operator &&

We can also call functions with JSX and embed the result of the calling function that return a value. You can include conditional statements inside your function or use ternary operator and ShortCircuit Operator &&


- Using Function Call

# Example 1 - Call a function that return a value with your JSX braces {} :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  function GetText(){
    return "Welcome Again"
  }
  
  const App = function () {
    return (
      <div>
        <h1> { GetText() } </h1>
      </div>
    );
  }; 
  
  ReactDOM.render(<App />, document.getElementById("root"));


  Welcome Again

# Example 2 - Using arrow function to return a string value :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  const callName = () => "Adam"
  
  function App() {
   const loading = false
  
   if (loading) {
    return <h3> Is Loading ....</h3>
   }
  
   return (
    <div>
     <h1>Hello {callName()} </h1>
    </div>
   )
  }
  
  ReactDOM.render(<App />, document.getElementById("root"))  


  Hello Adam

# Example 3 - Call a function that accept an object parameter and return from it FirstName and LastName :-

➤ An Example

- Using if-else

# Example 1 - Using if-else condition inside our function and return a value of user object :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  function formatName(user) {
    return user.firstName + " " + user.lastName;
  }
  
  function getGreeting(user) {
    if (user) {
      return <h1> Hello, {formatName(user)}! </h1>;
    }
      return <h1> Hello, Stranger. </h1>;
  }
  
  const user = {
    firstName: "Adam",
    lastName: "Frank",
  };
  
  const App = function () {
    return getGreeting();
  };
  
  ReactDOM.render(<App />, document.getElementById("root"));


  Hello, Stranger

# Example 2 - Using if-else condition inside our function and return a price value :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  const price = 8000;
  
  function Vehicles() {
    if (price < 5000) {		
      return <div>Car Price: {price + (price * 0.05)}</div>		
    } else {		
      return <div>Car Price: {price + (price * 0.15)}</div>		
    }
  }
  
  const App = function () {
    return Vehicles();
  };
  
  ReactDOM.render(<App />, document.getElementById("root"));


  Car Price: 9200

4 - Loops and Iteration For Loop and Map Method

# Example 1 - using map iteration method to loop thru an array :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  const cars = ["MG Hector", "Hyundai Venue", "Toyota Fortuner", "Hyundai Creta"];
  
  const carList = cars.map( (car) => <li key={car.toString()}> {car} </li> );
  
  function Vehicles() {
    return <ul> { carList } </ul>;
  }
  
  ReactDOM.render(<Vehicles />, document.getElementById("root"));


  MG Hector
  Hyundai Venue
  Toyota Fortuner
  Hyundai Creta

# Example 2 - using map iteration method to loop thru an array :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  function Numbers() {
    const elements = ["one", "two", "three"];
    return (
      <ul>
        {elements.map((value, index) => {
          return <li key={index}> { value } </li>;
        })}
      </ul>
    );
  }
  
  ReactDOM.render(<Numbers />, document.getElementById("root"));


  one
  two
  three

# Example 3 - using for loop for iteration :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  const elements = ["one", "two", "three"]
  
  const items = []
  
  for (const [index, value] of elements.entries()) {
   items.push( <li key={index}> {value} </li> )
  }
  
  function Numbers() {
   return <div> {items} </div>
  }
  
  ReactDOM.render(<Numbers />, document.getElementById("root"))


  one
  two
  three
