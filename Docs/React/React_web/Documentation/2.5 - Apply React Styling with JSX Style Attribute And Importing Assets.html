2.5 - Apply React Styling with JSX Style Attribute And Importing Assets

1 - What is JSX Styling ?

We have a style attribute which accept javascript object with camelCased properties rather than a CSS string. There are many ways to style React with CSS, we will take a closer look at inline styling, CSS stylesheet, CSS Modules and Styled Component. It's required to define CSS style properties with a camelCase format.

# Some CSS properties convert them in to JSX style properties :-


  background-color -> backgroundColor
  font-weight -> fontWeight
  max-width -> maxWidth
  padding-top -> paddingTop

2 - Using Inline Styling CSS In React

In order to define inline JSX style on your react element node, you need to pass the style prop/attribute a javascript object or reference to an object containing CSS properties and values. To style an element with the inline style attribute, the value must be a JavaScript object.

In the following example we are going to setup an inline styling with an object containing JSX inline styles using the {} bracket like this style={ { color: "red" } } defining our CSS properties with their values.

# An Example of inserting an style object within style attribute as inline styling :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"

  const App = () => {
    return(
      <div>
        <h1 style={{ color: "red", backgroundColor: "white" }}> Welcome to ReactJS </h1>
        <p> Information about your application </p>
      </div>
    )
  }

  ReactDOM.render(<App />, document.body)

React will transfer the JSX code using Babel and then parsed by a JavaScript engine, the resulting HTML will be as above.


  <h1 style="color: red; background-color: pink;"> Welcome to ReactJS </h1>
  <p> Information about your application </p>

In the example above we embedded our CSS as JSX inline styling object using {} brackets. As you see our CSS properties must be in a camelCase format it similar to when writing CSS properties in JavaScript. Since the inline CSS is written in a JavaScript object, properties with two names, like background-color, must be written with camel case as backgroundColor

3 - Using JavaScript Object Styling CSS In React

We can also create an object with styling information, and refer to it in the style attribute. We will create a variable object called MyStyles, containing jsx inline style with camelCase format, then pass MyStyles into JSX element tag with {} brackets to reference the object that should be used for the value of the style prop/attribute like this style={ MyStyles } let's see our example.

# An Example of inserting a style object within style attribute as object reference :-

➤ An Example


  import React from "react"
  import ReactDOM from "react-dom"
  
  const App = () => {

   const ImageURL = "./image/12512.png";

   const MyStyles = {
    color: "red",
    backgroundColor: "black",
    fontWeight: "bold",
    fontFamily: "Arial"
   }

   return (
    <div>
     <h1 style={ MyStyles }> Welcome to ReactJS </h1>
     <p> Information about your application </p>
     <img style={ { backgroundImage: 'url(' + ImageURL + ')' } } />
    </div>
   )
  }
  
  ReactDOM.render(<App />, document.body)


  <h1 style="color: red; background-color: black; font-weight: bold; font-family: Arial;> Welcome to ReactJS </h1>
  <p> Information about your application </p>
  <img style="background-image: url("./image/12512.png");">

React will transfer the JSX code using Babel and then parsed by a JavaScript engine, the resulting HTML will be as above.

4 - Using CSS Stylesheet In React

You can write your CSS styling in a separate file as an external CSS, just save the file with the .css file extension, and import it to your React components depending on where you need the styling to be applied. Let's create our external CSS file that define our css classes.

# Create a new file called "App.css" within the src folder, and insert some CSS code in it:

➤ ./src/App.css


  h1 {
    background-color: #282c34;
    color: white;
    font-family: Arial;
    text-align: center;
  }
   
  .button {
    cursor: pointer;
    border: 1px solid #1a202c;
    padding: 8px;
    min-width: 64px;
    background: transparent;
    transition: all 0.1s ease-in;
  }
   
  .button:hover {
    background: #1a202c;
    color: #ffffff;
  }

In order to use our App.css styling inside our react component all you need to do is import the CSS file into your component file, then use your CSS classes inside className attribute.

# Import the stylesheet in your application and you can use the CSS class right away within the className attribute :-

➤ ./src/App.js


  import React from "react"
  import ReactDOM from "react-dom"
  import "./App.css"  <-- Import Our External CSS file 
  
  const App = () => {
   return (
    <div>
     <h1>Hello Style!</h1>
     <p> Press the Button Below to continue </p>
     <button type="button" className="button">
      Click Here
     </button>
    </div>
   )
  }
  
  ReactDOM.render(<App />, document.body)

7 - Using CSS Modules In React

Another way of adding styles to your application is to use CSS Modules. A CSS Module is a CSS file in which all class names and animation names are scoped locally by default. When using CSS Modules, each React component is provided with its own CSS file, that is scoped to that file and component alone. That means CSS Modules are convenient for components that are placed in separate files.

A CSS Module happens at build time when local class names which can be super simple without conflict are mapped directly to the automatically-generated ones. If you check your browser inspect > Elements you going to find you class name with different name that are assigned by CSS Module at build time.

Creating the CSS Module is simple just add with .module.css before the CSS file name, for example MyStyles.module.css. So let's create our CSS Module.

# Create a new file called "styles.module.css" and insert some CSS code in it: :

➤ ./src/App.module.css


  .MyHeader {
    background-color: #282c34;
    color: white;
    font-family: Arial;
    text-align: center;
  }
  .MyParagraph {
    color: red;
   }
  .text-box note {
    color: rgb(255, 221, 2);
  }
  .MyButton {
    cursor: pointer;
    border: 1px solid #1a202c;
    padding: 8px;
    min-width: 64px;
    background: transparent;
    transition: all 0.1s ease-in;
  }
  .MyButton:hover {
    background: #1a202c;
    color: #ffffff;
  }   
  .MyFooter{
    color: aqua;
  }

Now to make use of CSS Module file inside our Component we need to import the module file directly into our App.js component and within the className we reference style like prop to access the styles as the following.

# Import the CSS Module file in your application and you can access the styles as a reference within the className attribute :-

➤ ./src/App.js


  import React from "react"
  import ReactDOM from "react-dom"
  import styles from "./App.module.css"
  
  function NoteText() {

  }
  function Footer() {
   return <p className="MyFooter"> Footer </p>
  }
  
  const App = () => {
   return (
    <div>
     <h1 className={styles.MyHeader}> Hello Style! </h1>
     <p className={styles.MyParagraph}> Press the Button Below to continue </p>
     <button type="button" className={styles.MyButton}>
      Click Here
     </button>
     <NoteText />
     <Footer />
    </div>
   )
  }
  
  ReactDOM.render(<App />, document.body)

8 - Using Styled-components Library In React

Styled-components is a library for React and React Native that allows you use component-level styles in your application that are written with a mixture of JavaScript and CSS. It was created with the same method of operation of CSS Modules, a way to write CSS that’s scoped to a single component, and not accessible to any other element in the page or even component.

Styled-components allows React developers to write plain CSS in React components without having to worry about clashing of class names. To use Styled-components library, first we need to install it by running the following command in your terminal.

# Install Styled-components :


> npm install styled-components
  @"styled-components": "^5.3.0"

Now we can import the styled component library into our component by writing import styled from 'styled-components';. then start writing or creating a variables and assign to them a particular HTML elements methods like div, p, span, ... and other more. After selecting our HTML element we open a back quote `` to store our style keys. Then we use the name of our variable we created as a wrapper around our JSX elements. Let's create our three styled components.

The CustomBtn is the styled component, and it will be rendered as an HTML button with the contained styles. styled is an internal utility method that transforms the styling from JavaScript into actual CSS. We can adapt styling using props. Each component accept props, we can assign a different background color based on the existence or value of the bg prop.

# Using props to adapt our styles by passing them thru our components to the internal styles :-

➤ ./src/App.js


  import React from "react"
  import ReactDOM from "react-dom"
  import styled from "styled-components";

  const StyledButton = styled.button`
    min-width: 200px;
    border: none;
    font-size: 18px;
    padding: 7px 10px;
    /* The resulting background color will be based on the bg props. */
    background-color: ${props => props.bgColor === "black" ? "black" : "blue";
  `;

  function App() {
    return (
      <div>
        <StyledButton bgColor="black"> Button A </StyledButton>
        <StyledButton bgColor="blue"> Button B </StyledButton>
      </div>
    )
  }

  ReactDOM.render(<App />, document.body)

9 - Importing Images and Other Assets

We can import files into our app components, like Images, styles or other assets. We have an image element that need a file that we can import from a folder that include all of our assets. We can import images files and other assets directly into our React components. This feature comes from Create React App's webpack configuration.

# Importing an image file from our src folder on the current folder :-

➤ ./src/App.js


  import React from "react"
  import ReactDOM from "react-dom"
  import "./App.css";
  import logo from "./logo.svg"; // the file exist on src folder or current folder

  function App() {
    return (
      <div className="App">
        <header className="App-header">
          <img src={logo} className="App-logo" alt="logo" />
          <h1> ReactJS Application </h1>
        </header>
      </div>
    );
  }

  ReactDOM.render(<App />, document.body)

We can instead of including or Importing our static assets directly within our src folder, we also have the option to include them in our public folder where index.html exist. And we can just use the file name with their extension like this "./image012.png, it will look on the public folder where index.html exist. All assets are stored with the assets folder which is located on the ./public folder. Let's see the following example

# Writing our string file path on the src attribute value directly instead of importing, it will start looking from the ./public/index.html path directory :-

➤ ./src/App.js


  import React from "react"
  import ReactDOM from "react-dom"
  import React from "react"
  import ReactDOM from "react-dom"
  
  const App = () => {
  
   return (
    <React.Fragment>
      <h1> Welcome To ReactJS </h1>
      <img src="./assets/images/image125749.jpg" />
    </React.Fragment>
   )
  }
  
  ReactDOM.render(<App />, document.getElementById("root"))
