1.1 - What's Express, and it's Features

- What is Express ?

Express is a web Framework based on the core Node.js HTTP module And libraries connect components called plugins or middleware.

Express is based on the Node.js middleware module called connect components, which in turn uses http module. So, any middleware which is based on connect will also work with Express.js.

Express provides a minimal interface with all the tools required to build a web application. It adds flexibility to an application with a huge range of modules available on npm that you can directly plug into Express as per requirement.

Express lets you structure a web application to handle multiple different http requests at a specific URL. It helps you respond to requests with route support so that you may write responses to specific URLs

- Express Features

Express solves these and many other problems. It provides ways to reuse code elegantly and provides a model-view-controller (MVC)-like structure for your web apps. In node one large request handler function for all requests, but in express each request has its own function

# Below I have picked out a few of the most important features of Express.js:

  • Express quickens the development pace of a web application.
  • Include third-party dependencies as well as your own modules such as controllers, utilities, helpers, and models
  • It also helps in creating mobile and web application of single-page, multi-page, and hybrid types
  • Define routes and their request handlers
  • Express can work with various template engines such as Pug, Mustache, and EJS.
  • Express follows the Model-View-Controller (MVC) architecture.
  • It makes the integration process with databases such as MongoDB, Redis, MySQL effortless.
  • Express also defines an error-handling middleware.
  • It helps in simplifying the configuration and customization steps for the application.
  • Start the app which will start the server on a particular host and port
