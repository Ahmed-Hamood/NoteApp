1.4 - Start Writing And Executing An Express Application

1 - Create A Simple Express Application

We have installed our express framework, now it is time to start developing our first app using Express. We will write a web server that runs locally on PORT 3000, So when you open http://localhost:3000/ URl in your browser, you should see Hello World. Now let's start writing our simple application

# Writing an express application which will listen on port number 3000 :

./server.js


  const express = require("express")
  const app = express()

  app.get("/", (req, res) => {
    res.send(" <h1> Hello World </h1> ")
  })

  app.listen(3000, () => {
    console.log("Listening on PORT 3000")
  })

- const express = require("express")

We are using the require function to include the express module into our project

- const app = express()

Now we can create an application object by instantiate an Express.js module

- app.get("/", (req, res) => { } )

We create HTTP GET Method for the home route o the first parameter. In the second parameter we have a request handler as a callback function, which will be called or invoked when the client hit the http://localhost:3000/ home root URL

- res.send("Hello World")

Inside the call back function we are responding and sending an HTML string text back to the client. The res parameter is used to send content back to the web page.

- app.listen(3000, () => { console.log("Listening on PORT 3000") })

Our server will listen to any request thru the PORT 3000. Our server application listen to client requests on port no 3000. You can specify any available port over here.

# Another way to startup a server using http :-

Click to show the full example ....

./server.js


  const express = require("express");
  const app = express();
  const http = require("http");
  
  app.get("/home" , (req, res) => { res.send("Hello World"); });
  
  http.createServer(app).listen(3000, function() {
    console.log("Express server listening on port 3000");
  });

2 - Run An Express Application

To run your express application, we execute node server.js from the project folder directory thru CMD (You may have to cd into the directory where server.js lives.). If everything works well you should see "Listening on PORT 3000" as a text appear on the screen

# Run your express application with current director terminal :-


> node server.js
Listening on PORT 3000
