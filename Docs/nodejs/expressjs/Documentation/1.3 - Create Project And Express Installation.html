1.3 - Create Project And Express Installation

1 - First You must have NodeJS JavaScript Runtime

Before installing Express into your system you need to make sure that you have Node.js already installed. If not then you have to visit this website NodeJS Installation visit this article to show you how to install nodejs in your system Article Link

# To confirm that node and npm are installed by running the following commands in your terminal :-


> node --v // -v short for --version
 v15.5.1
> npm --v 
 7.3.0

2 - Create Project folder

Create a project directory, and inside this project create a file call it index.js, app.js or server.js

# Project Content :-


> Project_Name ls
  └── server.js (File)

3 - Generate package.json

Once we are inside the project folder, we can create package.json manually in a text editor or with the npm init --yes in your terminal command. When you use this command, you’ll be asked to provide details such as project name, description, and so forth

# initialize npm to create a package json :-


> npm init // it's going to ask many questions 
 package name: (MyProject)
 ......

# Quicker way to create package.json :-


> npm init –-yes

# View the content of package.json :-

./package.json


  {
    "name": "MyProject",
    "version": "1.0.0",
    "description": "",
    "main": "server.js",
    "scripts": {
      "test": "echo \Error: no test specified\ && exit 1",
      "start": "node server.js"
    },
    "keywords": [],
    "author": "",
    "license": "ISC"
  }

# Project Content :-


> Project_Name ls
  └── package.json (File)
  └── server.js (File)

4 - Express Installation Using npm install express

Express.js is a node package manger (NPM) which is a dependency to your express application which works base on node.js. This means that every project that is build with express needs to have the express framework is a source file in the local node_modules folder not globally. you just need to install express like any other NPM module using npm install express

Express.js is a dependency module and should be installed in the local (project) node_modules folder using npm i express.

# If you want to install latest express version locally into your project folder Only, you need to execute the below command:


> npm install express
> npm install express --save 
-------------------------------------------------------
added 50 packages, and audited 51 packages in 4s
"express": "^4.17.1"
-------------------------------------------------------

You can save and install Express into your system, and whenever you need to use it in your newly created project, and start using express right away

# Install express.js package in your system To install it globally you can use the below command:


> npm install -g express

If you want to install latest Express.js for an existing project and save the dependency into the package.json file (smart thing to do!)—which is already present in that project’s folder—run:

All commands above will install the latest express version. If you want to install a specific express version use the following command

# Install a specific version for express:


> npm install express@4.8.1
> npm install express@4.8.1 --save

Now The package.json file with the latest Express.js version 4.17.1 dependency looks as follows:

# View the content of package.json :-

./package.json


  {
    "name": "MyProject",
    "version": "1.0.0",
    "description": "",
    "main": "server.js",
    "scripts": {
      "test": "echo \Error: no test specified\ && exit 1",
      "start": "node server.js"
    },
    "keywords": [],
    "author": "",
    "license": "ISC",
    "dependencies": { 
      "express": "^4.17.1" 
    }
  }
